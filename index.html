Tutorial: Jogo de Tiro ao Alvo (Godot 4.4.1)

Guia passo a passo.

Passo 1 — Configurações Iniciais

Antes de criar cenas e scripts, vamos garantir que o projeto esteja com as configurações corretas.

a) Janela / Resolução

  • Acesse Projeto > Configurações do Projeto > Janela.
  • Em Tamanho, defina: Viewport Largura = 1152 e Viewport Altura = 648. Essa resolução facilita cálculos e testes.

b) Mapear entradas (Input Map)

Abra Projeto > Mapa de Entrada e crie estas ações:

  • Ação: `mover_cima`
    • Vá para a aba Mapa de Entrada.
    • Na barra "Adicionar Ação", digite `mover_cima` e clique em Adicionar.
    • Clique no ícone de + ao lado do nome da ação.
    • Pressione a tecla `W` (ou a `Seta para Cima`) e clique em OK.
  • Ação: `mover_baixo`
    • Na barra "Adicionar Ação", digite `mover_baixo` e clique em Adicionar.
    • Clique no ícone de + ao lado da ação.
    • Pressione a tecla `S` (ou a `Seta para Baixo`) e clique em OK.
  • Ação: `mover_esquerda`
    • Na barra "Adicionar Ação", digite `mover_esquerda` e clique em Adicionar.
    • Clique no ícone de + ao lado da ação.
    • Pressione a tecla `A` (ou a `Seta para Esquerda`) e clique em OK.
  • Ação: `mover_direita`
    • Na barra "Adicionar Ação", digite `mover_direita` e clique em Adicionar.
    • Clique no ícone de + ao lado da ação.
    • Pressione a tecla `D` (ou a `Seta para Direita`) e clique em OK.
  • Ação: `shoot`
    • Na barra "Adicionar Ação", digite `shoot` e clique em Adicionar.
    • Clique no ícone de + ao lado da ação.
    • Pressione a tecla `Espaço` (ou clique com o `Botão esquerdo do mouse`) e clique em OK.

Passo 2 — Objetos: Alvo (Target) e Tiro (Bullet)

Vamos criar duas cenas simples: Target (Area2D com ColorRect vermelho) e Bullet (Area2D com ColorRect branco).

a) Cena Target

  1. Criar nova cena 2D → nó raiz: Area2D. Renomeie para Target.
  2. Filhos:
    • ColorRect (cor vermelha) — visual do alvo.
    • CollisionShape2D — ajustar para cobrir o ColorRect (RectangleShape2D).
  3. No painel Nó, adicione o grupo targets.
  4. Salve como Target.tscn e anexe o script Target.gd (abaixo).

Código do Target

Este script move o alvo para baixo e detecta quando ele é atingido por um tiro.

# Target.gd
extends Area2D

# Sinal para avisar o Main que este alvo foi atingido
signal hit

var speed: float = 120.0

func _physics_process(delta: float) -> void:
    # Move o alvo para baixo (entra pela parte superior da tela)
    global_position.y += speed * delta

func _on_area_entered(area: Area2D) -> void:
    # Se a área que entrou for do grupo bullets, emite sinal e se remove
    if area.is_in_group("bullets"):
        emit_signal("hit")
        queue_free()

Importante: Para que este código funcione, você precisa **conectar o sinal `area_entered`** do nó `Area2D` do alvo ao método `_on_area_entered` no editor(script).

b) Cena Bullet

  1. Criar nova cena 2D → nó raiz: Area2D. Nomeie Bullet.
  2. Filhos:
    • ColorRect (branco) — visual do tiro.
    • CollisionShape2D — adequar o retângulo (mais estreito que o alvo).
    • VisibleOnScreenEnabler2D — para remover o tiro ao sair da tela.
  3. No painel Nó, adicione o grupo bullets.
  4. Salve como Bullet.tscn e anexe o script Bullet.gd (abaixo).

Código do Bullet

Este script controla o movimento do tiro para cima e o remove ao atingir um alvo.

# Bullet.gd
extends Area2D

var speed: float = 600.0

func _physics_process(delta: float) -> void:
    # Move o tiro para cima (y decresce)
    global_position.y -= speed * delta

func _on_area_entered(area: Area2D) -> void:
    # Ao colidir com um alvo, se remove
    if area.is_in_group("targets"):
        queue_free()

func _on_visible_on_screen_enabler_2d_screen_exited() -> void:
    queue_free()

Importante: Você precisa **conectar o sinal `area_entered`** do nó `Area2D` do tiro ao método `_on_area_entered` no editor(script).

O tiro também precisa ser removido caso não acerte o alvo e saia da tela. Para isso, **conecte o sinal `screen_exited` do nó `VisibleOnScreenEnabler2D`** ao método `_on_visible_on_screen_enabler_2d_screen_exited()`.

Ambos os métodos contêm a chamada `queue_free()` para garantir que o nó seja removido da memória, o que é uma prática essencial para manter a performance do jogo.

Passo 3 — O Jogador (Player)

Criaremos a cena do jogador que pode se mover e disparar. Usaremos um ColorRect para o visual (cor azul).

a) Criando a cena do Player

  1. Nova cena 2D → nó raiz: CharacterBody2D. Renomeie para Player.
  2. Filhos do Player:
    • ColorRect (cor azul) — visual.
    • CollisionShape2D — escolha um RectangleShape2D que cubra o ColorRect.
  3. No painel Nó, adicione o grupo players.
  4. Salve como Player.tscn e anexe o script Player.gd (abaixo).

b) Script do Player — movimento e disparo

Explicação rápida antes do código: usamos Input.get_vector() para ler as quatro ações de movimento. O disparo tem um pequeno tempo de espera (cooldown) para evitar que o jogador atire continuamente.

# Player.gd
extends CharacterBody2D

@export var bullet_scene: PackedScene
var speed: float = 400.0
var can_shoot: bool = true
const SHOOT_COOLDOWN: float = 0.2

func _physics_process(delta: float) -> void:
	# Input.get_vector(left, right, up, down)
	var dir := Input.get_vector("mover_esquerda", "mover_direita", "mover_cima", "mover_baixo")
	velocity = dir * speed
	move_and_slide()

func _input(event):
	if event.is_action_pressed("shoot") and can_shoot:
		if bullet_scene:
			var bullet = bullet_scene.instantiate()
			# Adiciona o tiro à cena do jogo, ou seja, a mesma cena do Player
			get_parent().add_child(bullet)
			
			# Posiciona o tiro em relação ao jogador depois de adicioná-lo
			bullet.global_position = global_position + Vector2(0, -20)
			
			# Inicia o cooldown
			can_shoot = false
			await get_tree().create_timer(SHOOT_COOLDOWN).timeout
			can_shoot = true
		else:
			push_warning("bullet_scene não está atribuída no Inspector do Player.")

No Inspector do Player (na cena principal), arraste a cena Bullet.tscn para o campo bullet_scene.

Passo 4 — Lógica do Jogo e Colisões

Agora vamos criar a cena principal que vai gerar alvos, atualizar o placar e conectar sinais.

a) Criando a cena principal

  1. Nova cena 2D → nó raiz: Node2D. Salve como main_game.tscn.
  2. Filhos:
    • Player (instancie a Player.tscn).
    • Label chamado ScoreLabel (canto superior esquerdo).
    • Timer chamado TargetTimer — Wait Time = 1.0 e Autostart = true.
  3. No Inspector do nó principal, exporte a variável PackedScene e arraste Target.tscn para ela.

Código Principal

Este script instancia alvos, conecta sinais hit emitidos pelos alvos e atualiza o placar.

# Main.gd
extends Node2D

@export var target_scene: PackedScene
@onready var score_label: Label = $ScoreLabel
@onready var player: CharacterBody2D = $Player

var score: int = 0
var rng := RandomNumberGenerator.new()

func _ready() -> void:
    rng.randomize()
    score_label.text = "Pontos: 0"

# Nome da função ligada ao Timer (conecte o sinal timeout do TargetTimer)
func _on_TargetTimer_timeout() -> void:
    if not target_scene:
        push_warning("target_scene não atribuída no Inspector do Main.")
        return
    var t := target_scene.instantiate()
    add_child(t)
    # Posiciona o alvo em x aleatório e acima da tela (entra descendo)
    t.global_position = Vector2(rng.randi_range(50, 1100), -50)
    # Conecta o sinal custom 'hit' do alvo ao método abaixo
    t.hit.connect(_on_target_hit)

func _on_target_hit() -> void:
    # Chamado quando algum alvo emite 'hit'
    score += 10
    score_label.text = "Pontos: %d" % score

Importante: Para que o jogo funcione, você precisa **conectar o sinal `timeout()`** do nó `TargetTimer` ao método `_on_TargetTimer_timeout` no editor(script).

b) Fluxo de colisão resumido

  1. Player instancia Bullet (Player.gd).
  2. Bullet sobe e detecta o Target via area_entered.
  3. Ao acertar, Target emite sinal hit e se remove com queue_free().
  4. Bullet também se remove com queue_free() ao detectar o alvo.
  5. Main escuta hit e soma os pontos.

Depuração rápida: se o pontuador não aumentar, verifique (1) se o alvo está no grupo targets, (2) se o tiro está no grupo bullets, e (3) se você conectou o Timer e os sinais corretamente.

Passo 5 — Exportar e Dicas Finais

a) Exportando o jogo

  1. Vá em Projeto > Exportar....
  2. Adicione um preset (ex.: Windows Desktop ou Linux/X11).
  3. Se necessário, clique em Manage Export Templates para baixar os templates.
  4. Defina ícone, arquivos adicionais, e clique em Export Project.

b) Melhores práticas & ideias de evolução

Após terminar o tutorial, você pode evoluir seu jogo com estas ideias:

  • Pool de balas (object pooling) para melhorar a performance se houver muitos tiros.
  • Sons: adicione efeitos (disparo, acerto) usando AudioStreamPlayer2D.
  • Vidas/tempo: adicione um HUD com vidas, tempo restante ou níveis incrementais.
  • Substitua ColorRect por Sprite2D quando precisar de mais controle visual (animações, rotações, etc.).

c) Checklist final para rodar o jogo

  • Player com bullet_scene atribuído.
  • Target e Bullet adicionados a seus grupos (targets, bullets).
  • CollisionShape2D configurados corretamente.
  • TargetTimer autostart ativo e conectado ao Main.
  • ScoreLabel vinculado a $ScoreLabel no Main.
